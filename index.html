Welcome

DigiMusTh is a project about collecting german historical texts from the 19th century concerning musical theory. On this webpage you can therefore find the digitalized versions of various works. The project invites collaboration and you can find more information about adding a text yourself under documentation. To learn more about the project itself visit the about section. The team behind the project is introduced here.

This project is built upon previous work which can be found here:

  • "Digitizing the Dualism Debate: a case study in the computational analysis of historical music theory sources.
  • A minimal model for the mixed coding of text (TEI) and music notation (MEI)
